# Living Article Images

This directory contains images for articles in the Living category.

## Structure

- `/assets/images/living/[article-slug]/`
  - `hero.webp` (Main image)
  - `01-documents.webp`
  - `02-example.webp`
  - `03-checklist.webp`

## Guidelines

- Use `.webp` format for better performance.
- Follow the naming convention specified above.
- Do not use spaces or Japanese characters in filenames.
