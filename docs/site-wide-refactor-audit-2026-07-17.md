# Site-wide refactor audit — 2026-07-17

## Current implementation

- The public site is static HTML on GitHub Pages. Living, Events/News, and Learn German articles are generated from Markdown and registries.
- Community, Jobs, Eat, Shopping, and Medical use committed same-origin JSON generated by the Spreadsheet → GAS → GitHub Actions pipeline. Browser rendering does not call GAS in the normal path.
- The Pages artifact build generates stable Community and Jobs detail documents after copying the committed site. Current Jobs detail routes use `/germany/ja/jobs/{public-id}/`.
- The legacy Jobs query route remains noindex and redirects a valid active ID to its stable detail route after loading the same Jobs snapshot.
- Theme state is shared through `common.js`; shared layout is applied from the JA header/footer templates.

## Reverified observations

- Jobs listing/detail mismatch: resolved for current IDs. The current live listing exposes four active Jobs and links to stable, server-rendered detail routes. `/jobs/detail/?id=a` redirects to `/jobs/a/`; the historical `id=2` is not in the current public dataset and correctly renders a noindex not-found state.
- Brand identity: still open before this refactor. Page titles, social metadata, layout labels, legal/support copy, and fallback alt text use J-Connect Global as the Germany portal name.
- Shopping fallback: resolved as the normal live state. The current public Shopping snapshot contains 45 eligible records. Its guide remains an intentional empty/error fallback.
- Trust/content depth: already strong in Community, Living articles, Jobs posting, and many Events articles. These blocks should be preserved and aligned, not replaced.
- SEO foundations: canonical JA routes, JA/x-default hreflang, sitemap filtering, noindex utility/detail fallbacks, and structured-data validation already pass. EN/DE coming-soon routes are noindex and absent from hreflang and sitemap.
- Data safety: public-data generation uses allowlists and repository validation rejects private fields. Spreadsheet/GAS column names and endpoints are outside this refactor.

## Baseline verification

- Pages artifact: 5 Community and 4 Jobs details generated; 4 Jobs indexable.
- Production SEO/accessibility validator: 119 HTML files and 88 sitemap URLs passed.
- Static, layout, content, Jobs, public-data pipeline, and production-parity validators passed.
- Browser suite: 27/27 desktop, tablet, and mobile tests passed, including dark mode, overflow, Jobs states, directory states, legal pages, and generated details.

This audit records the pre-change state so the refactor can preserve working behavior while addressing the remaining shared root causes.
