# 銀行口座開設で使うドイツ語

## 使い方
銀行では、口座を開きたいこと、本人確認をしたいこと、必要書類を知りたいことを短く伝えます。難しい説明を一度に聞くより、手順を紙やメールで確認してもらうと進めやすくなります。

## 基本フレーズ

| 場面 | ドイツ語 | 日本語 |
| --- | --- | --- |
| 口座を開きたい | Ich möchte ein Girokonto eröffnen. | 普通口座を開設したいです。 |
| 予約したい | Ich hätte gern einen Termin zur Kontoeröffnung. | 口座開設の予約を取りたいです。 |
| 必要書類を聞く | Welche Unterlagen brauche ich? | どの書類が必要ですか。 |
| 本人確認 | Ich möchte meine Identität bestätigen. | 本人確認をしたいです。 |
| 住所証明 | Reicht die Meldebescheinigung als Nachweis? | 住民登録証明書は証明として足りますか。 |
| カード | Wann bekomme ich die Bankkarte? | 銀行カードはいつ届きますか。 |
| オンラインバンキング | Wie aktiviere ich das Online-Banking? | オンラインバンキングはどう有効化しますか。 |
| 手数料 | Gibt es monatliche Gebühren? | 月額手数料はありますか。 |

## 窓口での会話例

**Ich:** Guten Tag. Ich möchte ein Girokonto eröffnen.

**Bank:** Haben Sie einen Ausweis und eine Meldebescheinigung dabei?

**Ich:** Ja, meinen Pass habe ich dabei. Reicht die Meldebescheinigung als Nachweis?

**Bank:** Ja. Wir brauchen außerdem Ihre Telefonnummer und E-Mail-Adresse.

**Ich:** Verstehe. Wie aktiviere ich später das Online-Banking?

## オンライン銀行でよく見る言葉

| ドイツ語 | 意味 |
| --- | --- |
| Identitätsprüfung | 本人確認 |
| Ausweis | 身分証 |
| Meldebescheinigung | 住民登録証明書 |
| Kontoführung | 口座維持 |
| PIN | 暗証番号 |
| TAN | 取引認証番号 |
| Überweisung | 振込 |
| Lastschrift | 口座引き落とし |

## メール問い合わせ例

Betreff: Frage zur Kontoeröffnung

Sehr geehrte Damen und Herren,

ich möchte ein Girokonto eröffnen. Könnten Sie mir bitte mitteilen, welche Unterlagen ich brauche und ob ich einen Termin vereinbaren muss?

Vielen Dank im Voraus.

Mit freundlichen Grüßen

[Name]

## メモ

* パスポート、滞在許可、住所登録書類、税番号など、銀行によって求められる書類が違います。
* オンライン銀行では VideoIdent や PostIdent などの本人確認が出てくることがあります。
* 口座維持費、カード手数料、送金手数料を開設前に確認しましょう。
