Tax ID・Steuernummer・Steuerklasseの違い

ドイツ生活でよく出てくる税関連番号と税クラスの違いを、日本語で整理。

ドイツで働く、銀行口座を開く、確定申告をする、副業を始める、結婚後に税クラスを変更する。 そのたびに出てくるのが Tax ID / Steuer-ID / IdNr.、Steuernummer、Steuerklasse です。

名前が似ているため混乱しやすいですが、この3つはまったく別のものです。

最初にざっくり言うと、Tax ID は個人を識別する番号、Steuernummer は税務署での申告・事業に使う番号、Steuerklasse は給与から毎月どれくらい税金を差し引くかを決める分類です。

まず結論

ドイツ生活で最初に必要になる可能性が高いのは、ほとんどの場合 Tax ID です。

日本語ではまとめて「税番号」と呼ばれがちですが、実務では区別が重要です。

3つの違いを一言で言うと

Tax ID は「あなた個人の税務上のID」

Tax ID は、ドイツで税務上あなたを識別するための番号です。 正式には steuerliche Identifikationsnummer、略して IdNr. と呼ばれます。

特徴は次のとおりです。

  • 11桁の番号
  • 原則として一生変わらない
  • 引っ越しても変わらない
  • 管轄 Finanzamt が変わっても変わらない
  • 勤務先、銀行、税務署、家族給付などで使われる
  • 自分で選ぶものではなく、BZSt から割り当てられる

ドイツで会社員として働く場合、勤務先からまず聞かれるのは多くの場合この Tax ID です。

Steuernummer は「税務署との案件番号」

Steuernummer は、管轄の Finanzamt が税務処理のために割り当てる番号です。 Tax ID と違い、状況によって変わることがあります。

たとえば次のような場合に使われます。

  • 確定申告をする
  • フリーランス・個人事業を始める
  • Gewerbe を登録する
  • 請求書に税務番号を記載する
  • Finanzamt からの通知書に記載される
  • 税理士とやり取りする

会社員で、まだ確定申告をしたことがなく、副業もしていない場合は、自分の Steuernummer をまだ知らないことも珍しくありません。

Steuerklasse は「毎月の給与税を計算するための分類」

Steuerklasse は番号そのものではなく、給与から差し引かれる Lohnsteuer を計算するための分類です。

たとえば、独身なら通常 Steuerklasse 1、結婚している場合は 4/4、3/5、4 mit Faktor などの選択肢があります。副業・2つ目の雇用では Steuerklasse 6 が使われることがあります。

重要なのは、Steuerklasse は最終的な年間所得税そのものを決めるものではなく、毎月の給与からいくら前払い的に差し引かれるかに影響するものだという点です。

年末・確定申告で最終的な税額と差額が精算されるため、月々の手取りが多くても、あとで追徴が発生することがあります。

Tax IDとは？

Tax ID は、ドイツで税務上の個人識別に使われる11桁の番号です。 ドイツ語では次のように表記されます。

  • steuerliche Identifikationsnummer
  • Identifikationsnummer
  • IdNr.
  • Steuer-ID
  • Steueridentifikationsnummer

英語では Tax Identification Number、Tax ID、TIN などと呼ばれることもあります。

Tax ID が必要になる主な場面

Tax ID は、次のような場面で求められます。

  • 勤務先で給与計算を始めるとき
  • 銀行口座を開設するとき
  • Freistellungsauftrag を銀行に出すとき
  • 確定申告をするとき
  • Kindergeld など家族関連の手続きをするとき
  • Elterngeld、Mutterschaftsgeld などの申請で税務・所得情報が関係するとき
  • 年金・保険・行政機関とのやり取り
  • 税務署や ELSTER での本人識別

特に勤務開始時は、勤務先があなたの ELStAM、つまり電子的な給与税控除情報を取得するために Tax ID を必要とします。

Tax ID はいつ届く？

通常は、ドイツで初めて Anmeldung をしたあと、住民登録情報が税務当局側に連携され、BZSt から Tax ID の通知書が郵送されます。

流れはおおむね次のとおりです。

  1. Bürgeramt / Einwohnermeldeamt で Anmeldung をする
  2. Meldebehörde から BZSt へ情報が連携される
  3. BZSt が Tax ID を割り当てる
  4. 登録住所宛に郵送で通知が届く

そのため、Anmeldung をしていない場合、通常は Tax ID の発行プロセスが始まりません。

到着直後に注意したいこと

Tax ID の通知は原則として郵送です。 そのため、次の点が実務上かなり重要です。

  • 郵便受けに自分の名前を表示しておく
  • WG や仮住まいの場合、c/o 表記を確認する
  • 引っ越し直後なら Nachsendeauftrag も検討する
  • Anmeldung の住所と実際に郵便を受け取れる住所を一致させる
  • 会社・銀行に提出する前に番号を写真で保存しておく
  • 原本をなくさないように保管する

ドイツでは、郵便受けに名前がないと重要書類が届かないことがあります。 Tax ID、保険証、銀行カード、行政通知など、生活の基本書類は郵送で届くことが多いため、名前表示は早めに整えてください。

Tax ID が届かない場合

Anmeldung 後しばらく待っても Tax ID が届かない場合は、まず次を確認します。

  • Anmeldung は完了しているか
  • Anmeldung の住所に誤りがないか
  • 郵便受けに名前があるか
  • 同居人・大家・会社寮の郵便受けに届いていないか
  • 以前ドイツに住んでいたことがあり、すでに発行済みではないか

それでも届かない場合は、BZSt のオンラインフォームから再通知を依頼できます。 ただし、Tax ID は通常、電話やメールでその場で教えてもらえるものではありません。本人確認と個人情報保護のため、原則として登録住所宛に郵送されます。

Tax ID をなくした場合

Tax ID をなくしても、番号自体が消えるわけではありません。 次の書類に記載されていることがあります。

  • BZSt から届いた Tax ID 通知書
  • 過去の給与明細
  • Lohnsteuerbescheinigung
  • Einkommensteuerbescheid
  • Finanzamt からの書類
  • ELSTER 関連の情報

それでも見つからない場合は、BZSt に再通知を依頼します。

Tax ID は人に見せてもよい？

Tax ID は勤務先、銀行、税務署、行政機関など正当な理由がある相手には提出します。 ただし、SNS、掲示板、チャット、賃貸の内見相手など、必要性がはっきりしない相手にはむやみに共有しない方が安全です。

提出を求められた場合は、次を確認してください。

  • 何の手続きに必要なのか
  • 相手は勤務先・銀行・公的機関など信頼できる主体か
  • 他の番号と混同していないか
  • Steuernummer ではなく Tax ID を求められているのか

勤務開始時に会社へ提出するもの

ドイツで会社員として働き始めるとき、HR や Payroll から税務・社会保険関連の情報を聞かれます。

よく求められるのは次のような情報です。

Tax ID、Steuernummer、Sozialversicherungsnummer は別物です。 会社に聞かれたときは、どの番号を求められているのか確認してください。

Tax ID がまだ届いていないまま働き始める場合

ドイツ到着直後に勤務開始する場合、Tax ID がまだ手元にないことがあります。

この場合は、黙って放置せず、HR / Payroll に早めに伝えます。

伝えるべき情報は次のとおりです。

  • Anmeldung は完了しているか
  • Anmeldung の日付
  • Tax ID 通知を待っていること
  • 届き次第すぐ提出すること
  • 主たる勤務先なのか、副業先なのか

会社側は Tax ID がないと ELStAM を取得できないことがあり、その場合、給与計算上不利な Steuerklasse 6 が使われる可能性があります。 ただし、実際の処理は会社の Payroll、雇用形態、状況によって変わります。

Tax ID が届いたら、すぐに HR / Payroll へ提出してください。 多く引かれた税金がある場合、給与計算の修正や確定申告で調整される可能性があります。
