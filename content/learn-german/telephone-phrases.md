# 電話でよく使うドイツ語

## 使い方
電話では、名前、用件、希望する対応を最初に短く言います。聞き取れないときはすぐに聞き返し、必要ならメールで送ってもらいましょう。

## 電話の基本フレーズ

| 場面 | ドイツ語 | 日本語 |
| --- | --- | --- |
| 名乗る | Mein Name ist [Name]. | 私の名前は[名前]です。 |
| 用件を伝える | Ich rufe wegen eines Termins an. | 予約の件で電話しています。 |
| 担当者につないでもらう | Könnten Sie mich bitte mit Frau Müller verbinden? | Müllerさんにつないでいただけますか。 |
| 待つ | Ich bleibe in der Leitung. | このまま待ちます。 |
| 折り返し | Könnten Sie mich bitte zurückrufen? | 折り返し電話していただけますか。 |
| 電話番号 | Meine Telefonnummer ist ... | 私の電話番号は...です。 |
| 聞き返し | Könnten Sie das bitte wiederholen? | もう一度言っていただけますか。 |
| ゆっくり話す | Könnten Sie bitte langsamer sprechen? | もう少しゆっくり話していただけますか。 |

## 予約確認の会話例

**Ich:** Guten Tag. Mein Name ist Tanaka. Ich rufe wegen meines Termins an.

**Praxis:** Wann ist Ihr Termin?

**Ich:** Der Termin ist am Dienstag um 10 Uhr. Ich möchte den Termin bestätigen.

**Praxis:** Ja, der Termin ist eingetragen.

**Ich:** Vielen Dank. Könnten Sie mir die Adresse bitte per E-Mail schicken?

## 留守番電話に残すメッセージ

Guten Tag. Mein Name ist [Name]. Ich rufe wegen [Thema] an. Meine Telefonnummer ist [Nummer]. Bitte rufen Sie mich zurück. Vielen Dank.

## 電話でよく聞く表現

| ドイツ語 | 日本語 |
| --- | --- |
| Einen Moment bitte. | 少々お待ちください。 |
| Ich verbinde Sie. | おつなぎします。 |
| Die Kollegin ist gerade nicht da. | 担当者は今不在です。 |
| Rufen Sie bitte später noch einmal an. | 後でもう一度電話してください。 |
| Können Sie eine E-Mail schreiben? | メールを書いていただけますか。 |

## メモ

* 電話前に、名前、番号、用件、希望日時を紙に書いておくと安心です。
* 数字や日時は聞き間違えやすいので、最後に復唱しましょう。
* 長い説明が難しいときは “Kann ich Ihnen eine E-Mail schreiben?” と聞けます。
