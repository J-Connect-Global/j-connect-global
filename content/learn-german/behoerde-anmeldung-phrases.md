# 役所・Anmeldungで使うドイツ語フレーズ

## 使い方
役所では、短い文で用件をはっきり伝えると会話が進みやすくなります。事前に必要書類と質問を書き出し、複雑な内容はメールやメモで確認してもらいましょう。

このページはドイツ語表現のサポートです。手続き条件や必要書類は自治体によって異なるため、必ず居住地の公式サイトで確認してください。

## 基本フレーズ

| 場面 | ドイツ語 | 日本語 |
| --- | --- | --- |
| 予約がある | Ich habe einen Termin zur Anmeldung. | 住民登録の予約があります。 |
| 住民登録したい | Ich möchte meinen Wohnsitz anmelden. | 住所を登録したいです。 |
| 必要書類を聞く | Welche Unterlagen brauche ich? | どの書類が必要ですか。 |
| 書類が不足している | Mir fehlt noch die Wohnungsgeberbestätigung. | まだ住居提供者証明がありません。 |
| 後から提出できるか | Kann ich das Dokument nachreichen? | その書類を後から提出できますか。 |
| 書面で確認したい | Können Sie mir das bitte schriftlich geben? | それを書面でいただけますか。 |
| ドイツ語が得意ではない | Ich spreche noch nicht so gut Deutsch. | まだドイツ語があまり得意ではありません。 |
| ゆっくり話してもらう | Könnten Sie bitte etwas langsamer sprechen? | もう少しゆっくり話していただけますか。 |
| 証明書について聞く | Wo bekomme ich eine Meldebescheinigung? | 住民登録証明書はどこでもらえますか。 |
| 署名確認 | Muss ich noch etwas unterschreiben? | ほかに署名するものはありますか。 |

## 会話例

**Am Bürgeramt**

**Ich:** Guten Morgen. Ich habe einen Termin zur Anmeldung.

**Mitarbeiterin:** Haben Sie Ihren Pass und die Wohnungsgeberbestätigung dabei?

**Ich:** Meinen Pass habe ich dabei. Mir fehlt noch die Wohnungsgeberbestätigung. Kann ich das Dokument nachreichen?

**Mitarbeiterin:** Ja, bitte reichen Sie es per E-Mail nach.

**Ich:** Vielen Dank. Können Sie mir das bitte schriftlich geben?

## メール・問い合わせ例

Betreff: Frage zu den Unterlagen für die Anmeldung

Sehr geehrte Damen und Herren,

ich habe einen Termin zur Anmeldung am [Datum]. Könnten Sie mir bitte mitteilen, welche Unterlagen ich mitbringen muss?

Vielen Dank im Voraus.

Mit freundlichen Grüßen

[Name]

## メモ

* パスポート、Wohnungsgeberbestätigung、必要に応じて賃貸契約書を持参します。
* 必要書類や予約方法は都市ごとに異なります。
* 予約前に自治体公式サイトで最新情報を確認してください。
