# 買い物で使うドイツ語

## 使い方
買い物では、場所を聞く、在庫を確認する、会計で袋やカード支払いを伝える、返品したい理由を短く言う場面が多いです。A1の短い文だけでも十分に通じます。

## スーパー・ドラッグストアのフレーズ

| 場面 | ドイツ語 | 日本語 |
| --- | --- | --- |
| 場所を聞く | Wo finde ich Milch? | 牛乳はどこにありますか。 |
| 在庫確認 | Haben Sie das noch auf Lager? | これはまだ在庫がありますか。 |
| 値段確認 | Wie viel kostet das? | これはいくらですか。 |
| 商品を探す | Ich suche Shampoo für Kinder. | 子ども用のシャンプーを探しています。 |
| 袋がほしい | Eine Tüte bitte. | 袋を1枚お願いします。 |
| カード払い | Kann ich mit Karte zahlen? | カードで払えますか。 |
| レシート | Könnte ich den Kassenbon bekommen? | レシートをいただけますか。 |

## 会計での会話例

**Kasse:** Haben Sie eine Kundenkarte?

**Ich:** Nein, danke.

**Kasse:** Möchten Sie eine Tüte?

**Ich:** Ja, eine Tüte bitte. Kann ich mit Karte zahlen?

**Kasse:** Ja, bitte Karte einstecken.

## 返品・交換のフレーズ

| 場面 | ドイツ語 | 日本語 |
| --- | --- | --- |
| 返品したい | Ich möchte das zurückgeben. | これを返品したいです。 |
| 交換したい | Ich möchte das umtauschen. | これを交換したいです。 |
| 壊れている | Das ist leider kaputt. | 残念ながら壊れています。 |
| サイズ違い | Die Größe passt nicht. | サイズが合いません。 |
| レシートあり | Ich habe den Kassenbon dabei. | レシートを持っています。 |
| 返金方法 | Bekomme ich das Geld zurück? | 返金されますか。 |

## 店員さんに聞く質問

* Gibt es das auch in einer anderen Größe?
* Haben Sie das auch ohne Zucker?
* Ist das Angebot noch gültig?
* Wo ist die Kasse?
* Können Sie mir bitte helfen?

## メモ

* 袋は有料のことが多いので、必要なときだけ伝えます。
* 返品条件は店や商品によって違います。レシートを保管しておくと安心です。
* 分からない表示は写真を見せながら “Was bedeutet das?” と聞けます。
