# Kita・学校への問い合わせで使うドイツ語フレーズ

## 使い方
Kita、Tagesmutter、Großtagespflege、学校に問い合わせるときは、子どもの年齢、希望開始時期、希望する預かり時間、住んでいる地域を短く伝えます。空き状況、待機リスト、見学の可否を丁寧に確認しましょう。

このページはドイツ語表現のサポートです。入園や入学を保証するものではありません。申し込み方法は自治体や施設によって異なります。

## 基本フレーズ

| 場面 | ドイツ語 | 日本語 |
| --- | --- | --- |
| 空き状況 | Haben Sie freie Plätze? | 空きはありますか。 |
| 希望開始時期 | Wir suchen einen Betreuungsplatz ab ... | ...からの保育枠を探しています。 |
| 子どもの年齢 | Unser Kind ist ... Jahre alt. | 子どもは...歳です。 |
| 預かり時間 | Welche Betreuungszeiten bieten Sie an? | どの預かり時間がありますか。 |
| 待機リスト | Gibt es eine Warteliste? | 待機リストはありますか。 |
| 見学予約 | Können wir einen Besichtigungstermin vereinbaren? | 見学の予約を取れますか。 |
| 必要書類 | Welche Unterlagen benötigen Sie? | どの書類が必要ですか。 |
| 慣らし保育 | Wie läuft die Eingewöhnung ab? | 慣らし保育はどのように進みますか。 |
| 空きが出たら連絡 | Können Sie uns bitte informieren, wenn ein Platz frei wird? | 空きが出たら連絡していただけますか。 |
| 返信へのお礼 | Vielen Dank für Ihre Rückmeldung. | ご返信ありがとうございます。 |

## 問い合わせメール例

Betreff: Anfrage für einen Betreuungsplatz

Sehr geehrte Damen und Herren,

wir suchen einen Betreuungsplatz ab [Monat/Jahr]. Unser Kind ist [Alter] Jahre alt. Wir wohnen in [Ort/Stadtteil] und benötigen eine Betreuung von [Uhrzeit] bis [Uhrzeit].

Haben Sie freie Plätze oder eine Warteliste? Können wir einen Besichtigungstermin vereinbaren?

Vielen Dank für Ihre Rückmeldung.

Mit freundlichen Grüßen

[Name]

## 見学・Schnupperterminで聞く質問

* Welche Betreuungszeiten bieten Sie an?
* Wie läuft die Eingewöhnung ab?
* Gibt es Mittagessen?
* Welche Kosten entstehen zusätzlich?
* Gibt es Unterstützung, wenn unser Kind noch wenig Deutsch spricht?
* Was müssen wir zum ersten Tag mitbringen?

## メモ

* 空き状況は都市、施設、年齢、希望時間によって大きく変わります。
* 早めに問い合わせ、複数の施設を並行して確認すると進めやすくなります。
* 自治体ポータルや Jugendamt の案内がある場合は、そちらも確認してください。
